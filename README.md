# portela_bar
Criação de site para empresa

Este projeto destina-se à criação do site do bar e restaurante Portela Bar.
O projeto será desenvolvido por alunos do curso Tecnologia de Sistemas para Internt do IFB cujo ingresso ocorreu no 2º semestre de 2025. O projeto faz parte da disciplina Projeto Integrador I e também irá compor atividades de outras disciplinas ao longo do curso.
